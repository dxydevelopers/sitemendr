### **Typography System**

#### **Primary Font Family**
```

- Clean, modern, highly legible
- Excellent for UI and body text
- Weights: 300, 400, 500, 600, 700
```

#### **Hierarchy**
```
H1: Inter Bold 48px/56px (Hero headlines)
H2: Inter Semibold 36px/44px (Section headers)
H3: Inter Semibold 24px/32px (Card titles)
H4: Inter Medium 20px/28px (Subsections)
Body Large: Inter Regular 18px/28px (Lead paragraphs)
Body: Inter Regular 16px/24px (Main content)
Body Small: Inter Regular 14px/20px (Captions, metadata)
```

#### **Special Typography**
```
Code: JetBrains Mono (for technical content)
Accent: Inter Black (for emphasis/highlights)
```

### **Brand Guidelines**

#### **Logo Usage**
- **Minimum size**: 32px height
- **Clear space**: 1x logo height around logo
- **Color variations**: Full color, white, black
- **Don'ts**: Stretch, rotate, recolor, add effects

#### **Color Usage**
- **Primary**: AI Blue for main actions/branding
- **Secondary**: Tech Purple for accents/highlights
- **Neutrals**: Consistent across all applications
- **Accessibility**: WCAG AA compliance (4.5:1 contrast ratio)

#### **Typography Rules**
- **Line height**: 1.5x font size minimum
- **Letter spacing**: -0.02em for headlines
- **Text colors**: High contrast combinations only
- **Hierarchy**: Consistent scale across all platforms
