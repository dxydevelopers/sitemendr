# Core Sitemendr Business Operations

## What We Do (Simple Version)

### **Service 1: Bug Fixing & Technical Repairs**
```
Client Problem: "My website has bugs/errors"
Our Solution: Expert diagnosis and fixes
Pricing: $0 upfront (pay only if fixed) or hourly rates
Process: Assessment → Diagnosis → Fix → Testing → Delivery
```

### **Service 2: Custom Website Development**
```
Client Need: "I need a new website"
Our Solution: Full-service development from concept to launch
Pricing: Custom quotes based on project scope
Process: Consultation → Quote → Development → Launch → Support
```

## How We Work Day-to-Day

### **Client Acquisition**
```
Sources:
- Website inquiries (contact forms)
- Referrals from past clients
- Direct outreach and networking
- Online marketing (Google, social media)

Process:
1. Client contacts us with problem/need
2. Initial consultation call (30 minutes)
3. Assess requirements and provide quote
4. If accepted, move to project phase
```

### **Bug Fixing Workflow**
```
1. Client submits bug report with details
2. We assess and provide diagnosis (free)
3. Quote the fix cost and timeline
4. Client approves work
5. Fix the issue and test thoroughly
6. Deliver and provide 30-day warranty
```

### **Development Workflow**
```
1. Client consultation and requirement gathering
2. We create detailed project proposal and quote
3. Contract signing and deposit payment
4. Project kickoff meeting
5. Development in phases with client reviews
6. Testing and quality assurance
7. Launch and training
8. Post-launch support (30-90 days)
```
